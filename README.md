# ai-money-mentor

📌 Overview

AI Money Mentor is a simple and smart financial assistant designed to help people better understand their money. Many individuals struggle with financial planning because it feels complicated or expensive, so this project focuses on making it easy and accessible for everyone.

By entering basic details like income, expenses, and savings, users can quickly get an idea of their financial health along with helpful suggestions to improve it. The goal is to turn confusing financial decisions into clear and actionable steps.

🚀 Features
Calculates a financial health score based on user inputs
Provides simple and practical financial suggestions
Allows users to ask basic financial questions
Clean and easy-to-use interface for better user experience

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js (Express)
API: Simple REST API used to simulate AI-based responses
